# Project Grasswing

Ever wonder what if I could have the perks of newsjournals, music, notes and a homepage on a website? Fret not, Grasswing solves this issue. Built as both Project Andromeda and Project Campinas, Grasswing fuses both the design philosophy of "modern but functional" and the all-in-one experience. 

# Languages used

The project is being built on React JSX and Vite, coded in Zed. As of now, there are plans for a server based in SQL to preserve user data while using the project, specially in regards to the note system. 

# Contributions

Anyone can contribute to the project as long as it doesn't use Generative Artificial Intelligence or adjacents. If you have an idea and doesn't know how to code, suggest it and it will be listened as soon as possible.
